# Queue Cure

## Overview

Queue Cure is a real-time patient flow and digital queue platform designed to streamline hospital and clinic operations. It enables receptionists, doctors, and patients to manage consultations efficiently through live queue updates, QR-based access, and appointment scheduling.

## Features

- Real-time patient queue tracking using Socket.IO
- Automatic token generation
- QR Code for patient queue access
- WhatsApp queue link sharing
- Patient search by name or token
- Appointment booking
- Doctor Dashboard
- Receptionist Dashboard
- Analytics Dashboard
- Estimated waiting time calculation
- Missed and Cancelled patient handling
- Rejoin Queue functionality

## Tech Stack

- React.js

- CSS

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm start
```
